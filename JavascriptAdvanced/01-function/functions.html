JavaScript ECMA 6

Function

A linguagem Javascript possui meios de criar função por diversos meios. Na versão 6 foi introduzida a Arrow Function (=>).

Abaixo meios de criar funções no Javascript:

Function (modo legado)

Pelo modo legado é necessário indicar um nome para a função. Como boa prática, nomeie funções sempre com verbos (escrever, ler, testar...).

Exemplo de função:

function testar(value){
    console.log(value);
    }
    teste("Saudação!");

Resultado: Saudação!

Função Anônima

O Javascript permite adicionar uma função em uma variável. Quando isto é feito, o nome da função deve ser omitido.

var log = function(value){
    console.log(value)
    }
    log('Hallo!')

Resultado: Hallo!

Função com Return

Variável com uma função interna, retornando valor (utilização do RETURN)

    var log = function(value){
        return value+1;
    }
    console.log(log(1));

Resultado 2

Outra função anômina

Exibindo um exemplo de uma função de soma

    let sum=function(a,b){
        return a+b;
    }
    console.log(sum(1,2))

Resultado: 3

Arrow Function

Esta também é uma função anônima, portanto, nomeá-la é desnecessário, porém atribuí-la a uma variável é essencial.

    let sumArrow = (a, b) => a + b;
    console.log(sumArrow(3,4))

Resultado: 7

Arrow Function com statments

Quando uma Arrow Function for utilizada com statments, há a necessidade de se colocar {}.

    var arrowF = (a,b)=>{
        return a+b;
    }
    console.log(arrowF(4,6))

Resultado: 10

Arrow Function com um único parâmetro

Se a Arrow Function possuir apenas um parâmetro, os () poderão ser omitidos.

Importante salientar que Arrow Functions não podem ser utilizadas com Hoisting (função ser invocada antes de sua declaração).

    var elevarQuadrado = c => c*c
    console.log(elevarQuadrado(3));

Resultado: 9
